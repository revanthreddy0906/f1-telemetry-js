# Example templates

- `examples/vite-template`: quick start for Vite + React.
- `examples/next-template`: SSR-safe Next.js App Router integration.
- `examples/vue-template`: Vue starter using `@f1-telemetry-js/vue`.
- `examples/svelte-template`: Svelte starter using `@f1-telemetry-js/svelte`.
- `examples/react-native-template`: React Native starter using `@f1-telemetry-js/react-native-core`.
